# 🍉 Callback Array Processor (Node.js)

This is a fun and interactive Node.js console app that lets users:
- Convert a list of fruits to uppercase 🍎
- Double or triple a list of numbers 🔢

It's a simple exercise using **callbacks**, **user input**, and **array processing** with a bit of `setTimeout` magic ⏱️.

---

## ✨ Features

- Takes 3 fruits and 3 numbers from the user
- Offers 3 operations:
  1. Uppercase fruits
  2. Double numbers
  3. Triple numbers
- Uses callback functions and array manipulation
- Includes friendly prompts and validation

---

## 🛠️ Requirements

- [Node.js](https://nodejs.org/) installed
- Terminal (e.g., Command Prompt, PowerShell, Bash)
